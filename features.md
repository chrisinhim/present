1. Text box

   1. autocomplete-suggestions:

      1. autocomplete suggestion should be provided for the names of all the books of the bible: When the user partially writes the name of a book from the bible, then then the autocomplete suggestions should be provided.
      2. when the user selects a suggestion using arrows then then presses SPACE then the selected option should be entered in the text box.
      3. when the user does not select any suggestion and then presses SPACE, then the first suggestion should be entered in the text box.
2. Present button

   1. when there is no text entered
      1. show the message: "Please enter some text to show on the screen"
   2. when there is text
      1. Present the text on the Presentation window
   3. shortcut key - when the user hits ENTER key after entering the text in the text box, the text should be displayed on the presentation window.
   4. Position: Square button to the right of the text box.
3. Hide button

   1. shortcut key - ESC
   2. position: square button to the right of Present button.
4. Hold button
5. visible duration: this is the time duration for which the text on the presentation window will stay visible.

   1. in seconds
   2. reset on every Present
6. seek bar: This seek bar is a red color line below the text box. The seek will happen for the duration number of seconds. This will have a play & pause button in the beginning at the left. When the Present button is hit, the seek bar starts to seek and will complete in the number of seconds given for duration. When the pause button is clicked then the seek will pause.
7. History: History will be recording all the text that is displayed as line items with a Present history button in the end of the line and a Clear history item button at the beginning of the line.
8. Clear All History button:

   1. when there are history items, this button will delete all the history items.
   2. when there are no history items, this button will be inactive and greyed out.
9. Clear history item button

   1. This will delete it's own history item
10. Present history item button

    1. This button will present the text associated with it.
11. Open Presentation Window button

    1. This is a toggle button.
    2. This button will open the Presentation Window if it's not open.
    3. This button will close the Presentation Window if it's open
12. Font type:

    1. This option will be used to change the font style that is used for the text on the presentation window
13. font size

    1. This is the size of the text on the presentation window
14. font color

    1. This is the color of the tex on the presentation window
15. font gradient

    1. This is the gradient that is applied to the text on the presentation window
16. font style options

    1. This is the list of style options: Bold, Italic, underline, ALL CAPS, all small
17. Entry animation effects: These are the animation effects for the text when showing the text on the Presentation window

    1. None
    2. Slide: From left, from right, from top, from bottom
    3. Fade: Fade In
    4. Expand: horizantally, vertically, right, left, up, down
18. Exit animation effects: These are the animation effects for the text when removing the text from the Presentation window

    1. None
    2. Slide: From left, from right, from top, from bottom
    3. Fade: Fade Out
    4. Contract: horizantally, vertically, right, left, up, down
19. Presentation background solid color: This is the background color of the Presentation window
20. Presentation background picture: This is the background picture of the presentation window
21. Presentation background video: This is the background video loop of the presentation window
22. Text background toggle button: This toggle button wil enable/disable the text background
23. Text background type: picture (including transparent PNGs and animated GIFs), text line, box
24. Text background color: This is the background color for the text line or the box on the presentation window
25. Text background size: This is the size of the text background

    1. if the text background is picture, then it would be the size of the picture
    2. if the text background is text line, then it would be the width of the text line
    3. if the text background is box, then it would be the height and width of the box
26. Text background Decoration (Radius): This is the corner radius of the box
27. Text background width %:

    1. when the text background is selected as text line, then this is the width of the text line
    2. when the text background is selected as box, then this is the width of the box
28. Text background height %: This is the height of the box when the text background is selected as box
29. Text background picture: This is the option to select the picture with a browse button, when the text background is selected as picture.
30. Text Alignment horizontal options: These are the 3 options to aligh the text on the screen horizontally: left, center, right
31. Text Alignment vertical options: These are the 3 options to align the text on the screen vertically: top, middle, bottom
32. Text Alignment horizontal adjustment slider: This is a slider option to adjust the text horizontally from -50 to +50 where zero is the position set by the Text Alignment horizontal options.

    1. if the Text Alignment horizontal options is left, then the Text Alignment horizontal adjustment slider will only adjust the text until the mid of left and center.
    2. if the Text Alignment horizontal options is center, then the Text Alignment horizontal adjustment slider will only adjust the text until the mid of left and center in the left direction and until the mid of center and right in the right direction.
    3. if the Text Alignment horizontal options is right, then the Text Alignment horizontal adjustment slider will only adjust the text until the mid of right and center in the left direction.
33. Text Alignment vertical adjustment slider: This is a slider option to adjust the text vertically from -50 to +50 where zero is the position set by the Text Alignment vertical options.

    1. if the Text Alignment vertical options is top, then the Text Alignment vertical adjustment slider will only adjust the text until the mid of top and middle.
    2. if the Text Alignment vertical options is middle, then the Text Alignment vertical adjustment slider will only adjust the text until the mid of top and middle in the top direction and until the mid of middle and bottom in the down direction.
    3. if the Text Alignment vertical options is bottom, then the Text Alignment vertical adjustment slider will only adjust the text until the mid of bottom and middle in the top direction.
34. Preview: Preview display will display the same text at the top of the controls page. This preview will auto update as the user is typing
35. Auto Save: All the settings and values will be auto saved in the browser.
36. Save Design Button: This button is used to save the current settings as a file which can be loaded later
37. Load Design Button: This is a browse button that can load a previously saved design.
